* `src/`

    * `Environment`
    * `Firestorm`
    * `General Kocioraw`
    * `HearHound`
    * `Hero`
    * `LordRoyce`
    * `Minion`
    * `Miraj`
    * `TheCursedOne`
    * `TheRipper`
    * `Winterfell`
  * `game/` - contains implementation entrypoint class and basic gameplay structure
    * `Game`
    * `PrepareGame`
    * `Statistic`
  * `table/` - contains commands and player related classes
    * `Commands`
    * `Player`
